# Run Code

I used [vite](https://vitejs.dev) to make this project so...

Just clone the repository, and run:

```sh
npm i
```

and run any of this scripts:


```json
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview"
  },                            

```
```sh
npm run yourScript
```
